# MicroPython for ESP32

Welcome! 👋  
This repository contains code examples and exercises for learning **MicroPython on ESP32**.  
It's designed for students and anyone interested in experimenting with ESP32 projects.

---

## Usage

1. Install MicroPython on your ESP32.
2. Open an IDE like Thonny.
3. Upload the Python files to your board.
4. Follow the instructions in the code.
